# Sistema Escolhido

A Smart Fit é uma academia na qual usuários podem se cadastrar como clientes e instrutores.Nela,é possível inserir informações pessoais e técnicas (envolvendo os equipamentos).Além disso,ela apresenta planos de assinatura que vão desde o Padrão até o Premium e oferecem benefícios como ofertas exclusivas e acompanhamento com um Personal Trainer.
